# Todo App – C#

A simple console-based todo application built in C# to practice programming fundamentals.

## Features

- Add tasks
- View all tasks
- Remove tasks
- Input validation for empty tasks
- Safe number input using `int.TryParse()`
- Program logic separated into methods

## Technologies Used

- C#
- Console applications
- Lists (`List<string>`)
- Loops (`while`, `for`)
- Conditional statements (`if`, `switch`)
- Methods
- Input validation

## What I Learned

Through this project I practiced:

- Handling user input in console applications
- Working with lists and looping through data
- Structuring code into methods
- Improving code with input validation and error handling
